Engine Kinematics & Otto Cycle Simulator 사용방법

1. 기본 입력값

  • 보어(D): 실린더 지름을 mm 단위로 입력
  • 간극체적(Vc): 상사점에서 남는 체적을 cc 단위로 입력
  • 크랭크 반지름(r)과 커넥팅 로드(l): 피스톤 위치와 체적 변화를 결정

2. 각도 조절

  • 크랭크 각도 슬라이더를 움직이면 피스톤, 커넥팅 로드, L, L', 체적, P-V 선도의 현재점이 함께 이동
  • 본 시뮬레이터는 4행정 기준이며 각도 범위는 0~720°입니다.

3. 자동 재생

  • 자동 크랭크 회전을 체크하면 크랭크가 자동으로 회전
  • 재생 속도 슬라이더로 초당 회전 각도(°/s)를 조절 가능
  • 재생 중 점화 시점 근처에서 스파크 플러그 구현

4. 애니메이션 표시

  • 애니메이션 보이기를 체크하면 흡기 밸브와 배기 밸브의 열림/닫힘이 표시
  • 흡기 행정에서는 옅은 노란색 혼합기가 실린더 내부를 채우는 식으로 표시
  • 압축 행정에서는 혼합기가 실린더 안에 유지됩니다.
  • 폭발/팽창 이후에는 어두운 배기가 형성되고, 배기 행정에서는 회색·검은색 연기가 배기 포트 밖으로 빠져나감

5. 그래프 창 조작

  • 각 그래프 창의 상단 바를 드래그하면 위치를 옮길 수 있음
  • 오른쪽 아래의 +, -, Reset 버튼으로 엔진 그림의 확대/축소와 위치 초기화를 가능
